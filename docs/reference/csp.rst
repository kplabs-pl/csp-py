CSP.py Reference
================

``CspPacketFlags``

Defines flags for CSP packets.

- ``Zero``: No flags.
- ``Inherit``: Inherit flags from the parent packet.
- ``CRC32``: Use CRC32 for error checking.

``CspPacketPriority``

Defines priorities for CSP packets.

- ``Critical``: Critical priority.
- ``High``: High priority.
- ``Normal``: Normal priority.
- ``Low``: Low priority.
